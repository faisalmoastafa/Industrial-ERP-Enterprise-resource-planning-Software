Complete User Manual — Version 1.0

Welcome to Industrial ERP System — a full-featured desktop ERP application designed for industrial businesses. This manual walks you through every module so you can manage your operations confidently from day one.

Logging In

When you launch the application a splash screen appears while services start. After a few seconds the login page loads automatically.

Default Superadmin Credentials

FieldValue
Emailsuperadmin@erp.com
Passwordsuperadmin
RoleSuper Admin — full access to all modules
Security tip: After your first login, go to User Management → Edit Profile and change the default password to something strong and unique.

Steps to Log In

  1. Launch the application from the desktop shortcut or Start Menu.
  2. Wait for the splash screen to finish loading (usually 5–15 seconds).
  3. Enter your email and password on the login page.
  4. Click Login. You will be taken to the Dashboard.

Dashboard

The Dashboard gives you a real-time snapshot of your business. You will see:

  • Total Stats — overall sales, purchases, expenses, and income figures.
  • Month Overview — current month's sales vs. purchases chart.
  • Weekly Sales & Purchases — bar chart for the last 7 days.
  • Monthly Cash Flow — income vs. expense trend over 12 months.
  • Notifications — system alerts and low-stock warnings.
Dashboard widgets are permission-based. If a widget does not appear, your user role may not have the required permission. Ask the Super Admin to grant it.

Products & Inventory

The Products module is the heart of your inventory. Every item you buy or sell must first be created here.

Adding a New Product

  1. Go to Products → Products in the sidebar.
  2. Click the Create button (top right).
  3. Fill in the product name, code/SKU, category, unit, cost price, and selling price.
  4. Optionally upload a product image.
  5. Click Save.

Product Fields Explained

FieldDescription
Product NameFull descriptive name of the item.
Product CodeUnique SKU or barcode number.
CategoryGroup the product belongs to (create categories first).
UnitUnit of measurement (pcs, kg, litre, etc.).
Cost PriceWhat you pay to purchase this item.
Selling PriceWhat you charge your customers.
Alert QuantitySystem notifies you when stock falls below this number.

Managing Categories

Go to Products → Categories to add, edit, or delete product categories before adding products.

Printing Barcodes

Open any product and click Print Barcode. You can select the quantity to print and the barcode format (1D or 2D).

Stock Levels

Stock quantities are updated automatically when you record purchases, sales, adjustments, or manufacturing batches. You can always see the current quantity on the product list page.

Purchases

Record every item you buy from suppliers here. The system automatically updates your inventory stock when a purchase is saved.

Creating a Purchase Order

  1. Go to Purchases → Purchases.
  2. Click Create Purchase.
  3. Select the supplier from the dropdown (add them in People first if needed).
  4. Enter the purchase date and reference number.
  5. Add product lines — search for each product, enter quantity and price.
  6. Set the payment status (Paid, Partial, or Unpaid).
  7. Click Save. Stock is updated immediately.

Recording Payments

Open any existing purchase and click Add Payment to record partial or full payment to the supplier.

Purchase Returns

If you need to return items to a supplier, go to Purchases → Purchase Returns, create a return linked to the original purchase, and the stock will be decremented accordingly.

Sales & Billing

Record every sale you make to customers. The system automatically reduces stock and updates the customer's ledger.

Creating a Sale Invoice

  1. Go to Sales → Sales.
  2. Click Create Sale.
  3. Select or search for the customer.
  4. Add product lines with quantity and unit price.
  5. Apply any discount if applicable.
  6. Set payment status and method (cash, bank, etc.).
  7. Click Save. Stock is reduced automatically.

Printing an Invoice

Open any sale and click the Print button to generate a printable PDF invoice. The company name, logo, and footer are pulled from your Settings automatically.

Sale Returns

When a customer returns goods, go to Sales → Sale Returns and create a return linked to the original sale. Stock is added back and any refund is tracked.

Point of Sale (POS)

The POS module is designed for fast walk-in counter sales. It works with or without a barcode scanner.

Making a POS Sale

  1. Go to Sales → POS.
  2. Scan the product barcode OR search by product name in the search box.
  3. The item appears in the cart on the right. Adjust quantity if needed.
  4. Select the customer (optional — you can use a walk-in default).
  5. Enter cash received and the system calculates the change.
  6. Click Submit Order to complete the sale and print a receipt.
You can use a USB barcode scanner — just click in the product search box and scan. The product is added to the cart automatically.

Quotations

Send price quotes to customers before they confirm an order.

Creating a Quotation

  1. Go to Quotations in the sidebar.
  2. Click Create Quotation.
  3. Select the customer and add product lines.
  4. Click Save.

Converting to a Sale

Open any quotation and click Convert to Sale. The system creates a new sale invoice pre-filled with all the quotation details.

Emailing a Quotation

If email settings are configured, open the quotation and click Send Email to mail the PDF directly to the customer.

Customers & Suppliers

Manage all the people and businesses you trade with under the People section.

Adding a Customer

  1. Go to People → Customers.
  2. Click Create Customer.
  3. Enter name, phone, email, and address.
  4. You can also set an opening balance for existing debtors.
  5. Click Save.

Adding a Supplier

The process is identical — go to People → Suppliers and follow the same steps.

Viewing a Ledger

Open any customer or supplier profile and click View Ledger to see a full transaction history — all purchases, sales, payments, and outstanding balance.

Expenses

Track all outgoing costs that are not supplier purchases (rent, utilities, salaries, etc.).

Recording an Expense

  1. Go to Expenses → Expenses.
  2. Click Create Expense.
  3. Select the expense category (create categories first under Expenses → Categories).
  4. Enter the amount, date, and any notes.
  5. Click Save.

Income

Record non-sale income such as rental income, interest, or other revenue streams.

  1. Go to Income → Income.
  2. Click Create Income.
  3. Select the income category.
  4. Enter the amount, date, and reference.
  5. Click Save.

Payments & Ledger

All payments are recorded automatically when you create or edit purchases and sales. You can also manage them directly:

  • Sale Payments — go to Sales → Sale Payments to see all customer payments.
  • Purchase Payments — go to Purchases → Purchase Payments to see all supplier payments.
  • Open any customer or supplier profile for a full running ledger with balance.

Editing Amount Received

Users with the edit_sale_amount_received or edit_purchase_amount_received permission can directly edit the received amount on any transaction.

Returns

Both sale returns and purchase returns are supported. Returns automatically adjust stock levels and financial records.

TypeWhere to FindEffect
Sale ReturnSales → Sale ReturnsStock increases, customer credit recorded
Purchase ReturnPurchases → Purchase ReturnsStock decreases, supplier debit recorded

Manufacturing

The Manufacturing module lets you record the production of finished goods from raw materials.

Creating a Production Batch

  1. Go to Manufacturing in the sidebar.
  2. Click Create Batch.
  3. Select the finished product you are producing.
  4. Enter the quantity being produced.
  5. Add the raw material components and their consumed quantities.
  6. Click Save. Raw material stock is reduced; finished product stock increases.

Stock Adjustments

Use adjustments to correct stock levels due to damage, theft, counting errors, or stock-takes.

  1. Go to Adjustments in the sidebar.
  2. Click Create Adjustment.
  3. Select each product that needs correction.
  4. Enter the adjustment type: Addition to increase stock, Subtraction to reduce it.
  5. Enter the quantity and a reason note.
  6. Click Save.
Adjustments cannot be deleted after saving — they are permanent audit records. Always double-check quantities before saving.

HRM & Payroll

Manage your workforce — from hiring to monthly payroll.

Adding an Employee

  1. Go to HRM → Employees.
  2. Click Create Employee.
  3. Fill in personal details, department, designation, and base salary.
  4. Click Save.

Recording Attendance

Go to HRM → Attendances to mark employee attendance by date.

Managing Overtime & Bonuses

Go to HRM → Overtimes or HRM → Bonuses to record extra hours worked or bonus payments.

Running Payroll

  1. Go to HRM → Payrolls.
  2. Click Create Payroll.
  3. Select the employee and the payroll month.
  4. The system pre-fills salary, attendance deductions, overtime, and bonuses.
  5. Review and click Save.

Go to Settings → Units to create units like pcs, kg, litre, box, etc. These are used when creating products.

User Management

Control who can access the system and what they can do.

Creating a New User

  1. Go to User Management → Users.
  2. Click Create User.
  3. Enter the user's name, email, and a temporary password.
  4. Assign a role (Super Admin or Admin or any custom role).
  5. Set the account as Active.
  6. Click Save.

Roles & Permissions

The system ships with two default roles:

RoleAccess Level
Super AdminFull access to every feature including user management, backup, and system settings.
AdminFull operational access excluding user management, activity logs, and system utilities.

You can create additional custom roles and assign granular permissions to each one.

Activity Log

Super Admins can view a complete audit trail of all actions performed in the system by going to User Management → Activity Log.

Editing Your Profile

Any user can update their own name and password by going to User Management → Edit Profile.

Backup & Restore

Regular backups protect your data. The system stores all data in a local SQLite database file.

Important: Take a backup before performing major operations like bulk deletions, payroll runs, or system updates.

Creating a Backup

  1. Go to Settings → System Utilities → Backup.
  2. Click Create Backup.
  3. The system creates a timestamped ZIP file in your backup folder.
  4. The backup folder can also be opened directly from the File menu: Ctrl+B.

Restoring from a Backup

  1. Go to Settings → System Utilities → Restore.
  2. Select the backup ZIP file you want to restore.
  3. Click Restore. The application will reload with the restored data.

Backup Folder Location

Backups are stored in your Windows AppData folder:

%APPDATA%\industrial-erp\backups\

You can open this folder any time from the File → Open Backup Folder menu item.

Copy your backup files to an external drive or cloud storage (like Google Drive or OneDrive) for off-site protection.
